# E-commerce
Database-driven e-commerce platform focused on laptop and mobile accessories. Includes ER, UML, relational schema design, and MySQL implementation as part of a DBMS lab project.

1.Project Title:
 	E-Commerce Platform Exclusively for Laptops and Mobiles

2.Problem Statement:
	While online shopping is popular for its convenience and variety, existing e-commerce platforms are filled with unrelated products. This makes it difficult for customers to quickly find the right laptop and phone accessories. So this platform is a dedicated platform focused solely on accessories and can provide a streamlined shopping experience with accurate product details, easy navigation, and secure transactions .	

3.Project Objective:
	The main objective of this project is to design and develop a database-driven e-commerce website that:
•	Specializes in selling laptop and phone accessories.
•	Provides a structured product catalogue (phone cases, laptop cases etc.).
•	Ensures efficient search, filter, and recommendation features.
•	Supports user account management (registration, login, profile).
•	Handles shopping cart and order management and payment options

4.Scope of the Project:
I) Users/Customers: Can browse products, search, add to cart, and place orders.
II) Admin: Can manage product catalogue , view orders, update stock, and handle customer details.
III) Database: Stores product details, customer accounts, orders, transactions, and admin records.

5. Features
1.	User Features:
o	Registration and login.
o	Shopping cart and wish list.
o	Order placement
2.	Admin Features:
o	Add, update, or delete products.
o	Manage stock and pricing.
o	View customer details and purchase history.
o	Track sales reports.
3.	Database Features:
o	Normalized schema design for efficiency.
o	Use of primary keys, foreign keys, and stored procedures.
o	Ensuring data integrity and security.

6. Expected Outcomes
•	A fully functional e-commerce website prototype with database integration.
•	Structured and optimized database for managing product inventory and transactions.
•	A user-friendly interface that simplifies shopping for accessories.
•	Admin panel for effective business management.
7. Tools and Technologies
•	Front-end: HTML
•	Back-end: Python
•	Database: MySQL Client
